<strong>Healco</strong> es un proyecto que se creó para ser el sitio de una
empresa que ofrece productos y servicios orientados a la salud en las oficinas.

<hr />

<h1 class="text-2xl">Tecnologías y Herramientas</h1>
<ul>
  <li>
    <h6 className="text-lg font-semibold">Vue</h6>
    <p>
      Elegimos Vue como frontend framework para el proyecto ya que ofrecía una
      solución flexible y fácil de usar para nuestras necesidades. La sintaxis
      declarativa de Vue y la arquitectura basada en componentes lo hicieron
      ideal para nuestro proyecto.
    </p>
  </li>
  <li>
    <h6 className="text-lg font-semibold">Docker</h6>
    <p>
      Como se trataba de un proyecto en equipo y necesitábamos implementar el
      proyecto en un servidor, decidimos usar Docker para la implementación.
      Docker nos permitió crear un entorno en contenedores para el proyecto, lo
      que nos facilitó el mantenimiento del mismo entorno mientras lo
      desarrollábamos y durante la implementación.
    </p>
  </li>
</ul>

<hr />

<h1 class="text-2xl">Roles y Responsabilidades</h1>
El proyecto Healco fue mi primer proyecto completo como desarrollador. Fui parte
de un equipo y me ocupé principalmente del frontend, aunque también hice algo de
trabajo de backend.
<ul>
  <li>
    <h6 className="text-lg font-semibold">Componentes y vistas del frontend</h6>
    <p>
      Diseñé la parte del frontend del proyecto usando Vue, incluidos los
      componentes de la interfaz de usuario, los formularios y las vistas.
      También me ocupé del diseño y el estilo, y de la creación del sistema de
      navegación.
    </p>
  </li>
  <li>
    <h6 className="text-lg font-semibold">Estado y monitoreo del sistema</h6>
    <p>
      Participé en la creación del sistema de monitoreo y alerta del sistema,
      así como de la secuencia de comandos para la implementación del proyecto y
      la creación de la imagen de Docker.
    </p>
  </li>
</ul>
